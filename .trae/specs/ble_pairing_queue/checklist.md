# 队列式配对流程 - 验证清单

- [x] 检查 DesktopRelay 类是否添加了配对队列
- [x] 检查队列管理方法是否正确实现
- [x] 检查发现的设备是否能正确入队
- [x] 检查队列为空时是否能正确开始配对
- [x] 检查配对失败后是否能从队列取出下一个设备
- [x] 检查队列为空时是否能重新启动扫描
- [x] 检查配对成功后是否清空队列
- [x] 检查配对成功后是否停止扫描
- [x] 检查队列状态日志是否完善
- [x] 检查是否避免了竞态条件
- [x] 检查队列去重逻辑是否正确实现
- [x] 检查重复设备是否不会被添加到队列
- [x] 检查正在配对的设备是否不会被添加到队列
- [x] 检查重新扫描前是否清空队列
- [x] 检查重新扫描前是否清空失败设备列表
